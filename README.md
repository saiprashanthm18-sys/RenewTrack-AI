# RenewTrack AI ‚Äì Smart Renewable Energy Optimization System

RenewTrack AI is an AI-powered renewable energy optimization dashboard focused on the Indian energy landscape. It provides decision-makers with real-time insights, performance analytics, and predictive modeling for solar and wind energy assets.

## üöÄ Features

1.  **Dashboard Overview**: High-level KPIs including Installed Capacity, Generation, and CO2 reduction metrics.
2.  **Utilization Analysis**: Identification of underperforming assets and state-wise efficiency rankings.
3.  **AI Prediction Module**: Machine Learning based forecasting using weather parameters (Linear Regression).
4.  **CO2 Reduction Calculator**: Environmental impact simulator based on efficiency gains.
5.  **AI Recommendation Engine**: Smart decision cards with actionable interventions for low-performing states.
6.  **India Heatmap**: Interactive geographic visualization of energy efficiency across key states.
7.  **Scenario Simulator**: Strategic "what-if" modeling for national impact projection.

- **Frontend/Backend**: [Streamlit](https://streamlit.io/)

- **Theming**: Custom CSS (Sustainability Green/Blue)

3. **Run the application**:
